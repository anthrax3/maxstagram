Maxstagram

Maxstagram uses randomness to generate image filters.



Filtered images are generated in batches of 50 and the links to download them are emailed to you. If you like any of the results, you can download a high-quality version for free. Most of the results will be awful, but you do find the occassional gem that might make a nice wall decoration. Happy hunting!

The source code for this entire site is available at Github.
